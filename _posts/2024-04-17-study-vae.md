<h3>Concept Summary</h3>

+ VAE consists of an encoder that learns q(z|x), z, and a decoder that learns p(x|z), with the reparametrization trick applied between the encoder output and decoder input.
  + The Encoder performs dimensionality reduction (downsampling) on images or input data to learn the distribution of input features (z)
  + The Decoder performs upsampling from the feature distribution to learn reconstruction of the input features
+ Reference: [https://velog.io/@iissaacc/Upsampling](https://velog.io/@iissaacc/Upsampling)

> **Terminology**
+ z: Discrete latent random variable (latent space)
+ q(z∣x): Posterior distribution of input x (distribution of z given x)
+ p(z): Prior distribution
+ p(x∣z): True distribution of input x (distribution of x given z)

<h3>Vector Quantization</h3>

![VQ-VAE](/assets/img/vae/vq-vae.png)
VQ-VAE adds a discrete codebook to the Autoencoder structure. A codebook is essentially a list containing vectors as elements. When the encoder outputs a vector, the distances to all vectors in the codebook are calculated. Among the codebook vectors, the one closest to the encoder output vector is found. That vector is then fed into the decoder for training.

By adding vector quantisation (VQ) to VAE, p(z) is learned as discrete latent variables, and the embedding vector extracted from p(z) passes through the decoder.

__prior__
In the VQ-VAE paper, the prior distribution for discrete latents p(z) is a categorical distribution, which can be made autoregressive depending on other z values in the feature map.
This paper uses PixelCNN for images and WaveNet for raw audio.
