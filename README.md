## Local Development

### Prerequisites
- Ruby 3.x
- Bundler
- Jekyll

### Setup

```bash
# Clone the repository
git clone https://github.com/akillness/akillness.github.io.git
cd akillness.github.io

# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve

# Open in browser
open http://localhost:4000
```

### Creating New Posts

```bash
# Create a new post
touch _posts/YYYY-MM-DD-your-post-title.md
```

Post front matter template:

```yaml
---
title: "Your Post Title"
description: "Brief description for SEO"
categories: [Category1, Category2]
tags: [tag1, tag2, tag3]
date: YYYY-MM-DD HH:MM:SS +0900
mermaid: true
math: true
image:
  path: /assets/img/path/image.png
  alt: "Image description"
---
```
